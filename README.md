# 🐣 Easter Company Developer Environment

Welcome to **EDE** with Darwin first class integration, the next-generation development environment from Easter Company. Darwin is not just a text editor; it's a unified ecosystem designed to empower developers with cutting-edge AI assistance.

## 🚀 The Darwin Experience

- **AI-First Design**: Seamless integration with the **pi** coding agent.
- **Unified Branding**: A consistent, professional aesthetic for all your development tools.
- **Blazing Fast**: Pre-configured Neovim (LazyVim base) for zero-latency coding.
- **Smart Monitoring**: Real-time AI tool-call monitoring (Agent Monitor).
- **Easter Company Ecosystem**: Part of the broader EC software suite.

## 📦 Installation

Experience Darwin today with a single command:

```bash
curl -fsSL https://raw.githubusercontent.com/EasterCompany/EDE/main/install.sh | bash
```

## ⌨️ Global Shortcuts

| Shortcut | Action |
| --- | --- |
| `<C-e>` | Toggle File Explorer |
| `<C-\>` | Toggle Pi AI Agent |
| `<leader>ps` | Restore Agent Monitor |
| `<leader>gg` | Open Lazygit |
| `<leader>f` | Find Files |

## 🛠️ Included Components

1.  **Darwin Neovim**: A custom-branded Neovim configuration built on LazyVim.
2.  **Pi Agent Extension**: Darwin-specific intelligence for your AI coding assistant.
3.  **Darwin CLI**: The `darwin` command, your entry point to the IDE.
